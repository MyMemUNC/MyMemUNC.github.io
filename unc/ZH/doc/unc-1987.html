⚠ 限制訪問（101）

UNC-1987-C 藍面弄臣「製作你自己的香煙吧！」
威脅程度 C
收容/控制難度
C
實驗規條

C-N-1987 目前被特許供 C 級或以上人員於 Site-28 地下三層休息室內使用。除涉及高風險異常成分外，使用無須提前報備。所有投入口內容須經初步篩選，確認所選用之混入物質符合「人類可吸收」定義，並不得混入基金會列為禁用品之物質（包括但不限於認知危害物、強烈放射性元素與高等級異常碎片）。


實驗性混入行為需由 B 級人員或以上級別申請，並交由相關倫理小組進行審查。每次使用可自選單一或混合物質進行捲製。根據測試結果，建議每日使用不超過三根以防止攝入過量。

儲存規條

C-N-1987 與其附屬裝置被收容於 Site-28 地下三層的██號儲存櫃內。物資部門需遵循標準清潔與補給程序，每日對 C-N-1987 進行清潔，並補充預製煙料包。每週自動執行一次內部維護程序，該程序無需人員介入。


為便於研究與風險控制，各層休息區配備有用於營養與副作用評估的預製煙。儘管目前監測數據顯示 C-N-1987 所生成香煙未含已知成癮物質，但實驗數據表明仍存在輕微心理依賴傾向。為此，建議人員每日使用量不超過三根，以降低吸收過量反應及微成癮習慣形成的風險。所有使用記錄需定期提交至研究部門以供分析。

描述

藍面弄臣『製作你自己的香煙吧！』（下稱C-N-2187）為一異常製煙裝置，高約 1.6 公尺。裝置正面設有單一投入口與煙支輸出口，具備自動捲煙機械結構與自供能源系統，無需外部電力供應，內部結構異常複雜，目前尚無法完整拆解重現。整體外觀類似傳統販售機，以藍白條紋塗裝，表面印有一張受損之人形面具圖像，推測即名稱中之「藍面弄臣」。


使用者可將各類可被人類吸收之有機或無機物質投入投入口，C-N-2187 將進行自動轉換程序，平均 15～20 秒內輸出一根成品香煙，製煙完畢後由機體吐出煙支時所發出的固定語音播報「祝你吸得愉快！」，已確認為無異常認知效果之錄音，屬非主動訊息輸出。。該香煙可正常點燃，具標準燃燒特性與穩定煙霧釋放，無燃燒異味或雜質。裝置對煙霧味道與物理形質具適配能力，煙霧將依據所用原料呈現對應特徵，無異常擴散行為。


C-N-2187 所製之煙支具有顯著的生物吸收效率。人體吸入煙霧後，可攝取超過原物質 90% 以上的營養成分（包括微量元素、蛋白質衍生物、合成胺基酸及部分脂溶性維生素等），但需特別指出的是，吸食行為並未使人體攝取任何可量化的熱量或能量單位。尚未觀察到煙霧中保留有可引發急性毒性反應的殘餘化學成分。


若使用者在短期內吸入過量香煙（例如單一配方每日超過八根），將可能導致體內營養失衡與代謝紊亂，表現為輕度頭痛、內分泌失衡、代謝混亂、神經性錯覺、神經敏感度下降、胃腸機能干擾等生理異常。症狀於停止使用 48–72 小時內可自然恢復。


裝置製成之香煙即使未混入任何具有成癮性之物質，仍顯示輕微心理依賴性。成癮機制推測與香煙氣味、吸收反應所誘發之短期神經獎賞回路有關。該依賴性強度低，平均僅需七日內即可戒除，且無戒斷期症候。根據現行規則與調查結果，C-N-2187 所製香煙可視為合法的調節性異常補充物，准予人員於休息時段自由使用。
